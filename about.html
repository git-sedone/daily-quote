Daily quote

This project is coded in vanilla Javascript, this is my first project and I'm very proud of it. It returns a random quote from the 'quote garden' API each time the get quote button is pressed.



Daily quote works both in desktop and mobile versions;

The desktop version has a top navigation menu' where the user can choose between various options included:
  • Registration form, with all the input fields validated

  • About page where users can find information about this project

    • and finally a:

  • Contacts page where I displayed all my contact details
